# bilingual-todo
Angular Multilanguage Todo sample

# Tech Stack

- Angular 11.x
- Google firebase
    - Authentication
    - Could Firestore
- Angular Material 11.x
- Bootstrap 4.5 (Optional)
- TypeScript 4.x
- Sass or Scss
- Angular i18n ([https://angular.io/guide/i18n](https://angular.io/guide/i18n))
- RxJS
- NgRx (Optional)
